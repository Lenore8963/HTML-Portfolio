Projects

  • Doggo Mingo Website

    • Doggo is a social platform designed to connect pet lovers, aiming to enhance community engagement through features like user profiles, dog showcases, and interactive posts. As the sole developer, I designed and built both the frontend and backend of the application.
    • I used React.js for the frontend, ensuring a dynamic and responsive user interface, and Tailwind CSS for styling, which provided a seamless experience across different devices. For the backend, I used Node.js and Express.js to handle server-side logic and MongoDB for data storage due to its scalability and speed.
    • Additionally, I integrated OpenAI for intelligent search capabilities. AWS S3 was utilized for reliable image storage, ensuring high durability and availability of user-uploaded content. I implemented user authentication using JWT to secure user data and sessions. To streamline the development and deployment process, I set up a continuous integration and deployment (CI/CD) pipeline using GitHub Actions and Heroku. These implementations led to a robust, user-friendly platform that successfully engages the pet lover community.

  • Vancouver Real Estate Prices Analysis

    • As the leader of a team, I spearheaded the development of a Decision Tree regression model in Python to predict Vancouver real estate prices, leveraging the power of machine learning to provide accurate property valuations. By implementing and visualizing data using Python libraries such as Pandas, Matplotlib, and Seaborn, I ensured a comprehensive analysis and clear presentation of the data trends and model performance.
    • To make the predictions accessible and user-friendly, I led the creation of a web application using Streamlit. This real-time platform allows users to input property details and instantly receive price predictions, enhancing user experience and engagement.
    • The model achieved an impressive R-squared value of 0.88, indicating strong predictive performance and reliability. This project not only showcased my skills in data science and web development but also demonstrated my leadership abilities in guiding a team to build effective tools that deliver practical value to users.

  • Pawlog Mobile App

    • As a key member of the development team, I contributed to building a React Native mobile application integrated with Firebase for database management and user authentication. This project aimed to enhance the daily management of dog care routines by leveraging modern mobile technologies.

Our interactive map feature allows users to track and visualize their walking routes, enhancing the overall user experience.
    • Additionally, we incorporated a notification system to foster community engagement by sending timely reminders and updates. This comprehensive application not only streamlined dog care routines but also strengthened the bond within the pet-loving community through enhanced interaction and engagement.
